# Инструкция по обучению самообучающейся нейросети

## Принцип работы

Модель использует предобученные подмодели для детекции объектов (мебели, стен и т.д.) и обучается на основе пользовательской оценки (0-5), где:
- 0 - максимально плохо
- 5 - максимально хорошо

## Процесс обучения

1. Модель генерирует 2D-план на основе изображения
2. Вы оцениваете результат от 0 до 5
3. Модель использует вашу оценку как сигнал для улучшения будущих предсказаний
4. Процесс повторяется, и модель постепенно улучшает качество генерации планов
