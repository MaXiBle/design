# Нейросеть реконструкции комнаты

## Обзор
Компонент `RoomReconstructor` использует симуляцию нейросети для преобразования фотографии комнаты в 2D реконструкцию. Нейросеть анализирует изображение и создает текстурированное 2D представление комнаты с приблизительными размерами и расстановкой объектов.

## Архитектура нейросети

### RoomReconstructionNeuralNetwork
Основной класс, представляющий нейросеть для реконструкции комнаты:

- `loadModel()`: Загружает обученную модель (в симуляции - инициализирует параметры)
- `reconstructRoom(imageData)`: Принимает изображение и возвращает 2D реконструкцию
- `processImage(imageData)`: Обрабатывает изображение и извлекает информацию о комнате
- `analyzeRoomPart(imageData)`: Определяет, какая часть комнаты видна на изображении
- `generateTextures(analysis)`: Создает текстуры стен, пола, окон и дверей
- `generateObjects(analysis)`: Генерирует объекты мебели на основе типа комнаты

## Функциональность

### Анализ изображения
Нейросеть определяет:
- Тип комнаты (гостиная, спальня, кухня, ванная)
- Видимые стены
- Наличие окон и дверей
- Приблизительные размеры

### Генерация 2D модели
На основе анализа нейросеть создает:
- Текстуры стен с реалистичными цветами
- Пол с подходящей текстурой
- Окна и двери в правильных местах
- Мебель, соответствующую типу комнаты

## Поддерживаемые типы комнат
- Гостиная
- Спальня
- Кухня
- Ванная комната

## Точность и доверие
Нейросеть возвращает уровень доверия к реконструкции (в процентах), что позволяет пользователю понимать, насколько точной является реконструкция.

## Использование
1. Пользователь загружает фото комнаты
2. Нейросеть анализирует изображение
3. Создается 2D реконструкция с текстурами и объектами
4. Пользователь может видеть тип комнаты и уровень доверия

## Реализация
Текущая реализация является симуляцией. В продакшен-версии это будет заменено на реальную нейросеть с использованием TensorFlow.js или аналогичной библиотеки машинного обучения.
